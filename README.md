# Pickupable Timepieces

A mod that allows you to pick up calendars and clocks found throughout the world.

## Features

- Pick up wall clocks.
- Pick up calendars.
- Preserves the vanilla appearance and functionality.

## Installation

1. Download the latest release.
2. Extract the `pickupable_timepieces` folder into your `Mods` directory.
3. Launch the game.
